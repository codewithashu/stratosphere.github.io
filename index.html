What is Stratosphere?

A Project
Stratosphere is a project whose goal is to develop the next generation Big Data Analytics platform.
A Big Data Analysis Platform
Stratosphere was also the name of the system that the project had developed and is now entering the Apache Software Foundation through the Apache Incubator. The system is now developed by the Apache Flink project. Go to the project page at flink.incubator.apache.org.
